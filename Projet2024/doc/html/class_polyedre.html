◆ computeConvexity()

void Polyedre::computeConvexity ( )

Calcul si le polyedre est convexe ou pas.

Pour savoir si un polyedre est convexe, le principe est de verifier si pour chaque face, tous les sommets sont du meme cote. Cette methode met a jour un attribut qui est accessible avec la fonction isConvexe (pour eviter de refaire le calcul)
